<h1> نحوه استفاده </h1>
این مخزن شامل تعدادی نمونه کد به زبان اسمبلی IBM x390x می‌باشد.<br/>
در پوشه tools، اسکریپت ibm وجود دارد که با آن می‌توانید کد خود را اجرا کنید. نحوه اجرا کردن با کمک این اسکریپت به صورت زیر است:
<div dir="ltr">
  ./ibm code
</div>
دستور بالا فایل code.s را اجرا می‌کند.<br/>
این پوشه شمل فایل macro.s نیز می‌باشد که شامل چند ماکرو کاربردی است و می‌توانید از آن‌ها استفاده کنید یا در صورت نیاز آن‌ها را تغییر دهید.<br/>

<h1> برنامه‌های پیشنیاز </h1>
برای اجرای برنامه های به زبان اسمبلی s390x به qemu نیاز داریم.
این برنامه را با کمک دستور زیر می‌توانید دریافت کنید:    
<div dir="ltr">

  sudo apt install build-essential gcc-s390x-linux-gnu qemu-user

</div>

<h1> نحوه استفاده </h1>
این مخزن شامل تعدادی نمونه کد به زبان اسمبلی IBM x390x می‌باشد.
در پوشه tools، اسکریپت ibm که با آن می‌توانید کد خود را اجرا کنید وجود دارد. نحوه اجرا کردن با کمک این اسکریپت به صورت زیر است:
<div dir="ltr">
  ./ibm code
</div>
دستور بالا فایل code.s را اجرا می‌کند.
این پوشه شمل فایل macro.s نیز می‌باشد که شامل چند ماکرو کاربردی است و می‌توانید از آن‌ها استفاده کنید یا در صورت نیاز آن‌ها را تغییر دهید.
